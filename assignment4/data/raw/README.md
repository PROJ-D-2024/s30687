## Rules

1. Do not edit raw files in place.
2. If a file is replaced with a newer version, keep a note about the source and date.
3. Large or private raw source files are normally not tracked in Git; only the tiny synthetic sample bundle is committed for reproducibility.
4. If more thesis datasets become available, add them here and update the config and documentation accordingly.
5. The default reproducibility path uses the committed synthetic sample bundle; real thesis inputs can replace it later through configuration.
