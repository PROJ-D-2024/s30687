# Raw data

Place immutable source files here.

## Expected file for the current reproducibility scaffold

- `gp_practice_population_demographics_merged.xlsx`

## Rules

1. Do not edit files in place.
2. If a raw file must change, keep the original version and document the source/date.
3. The repository intentionally does not track raw Excel data in Git.
4. If additional thesis datasets become available, place them here and update `config/base.yaml` plus `docs/reproducibility_protocol.md`.
