# Data organization

This directory separates raw and processed data to preserve reproducibility.

## Structure

- `raw/` — original downloaded files, never edited manually
- `processed/` — cleaned and merged analytical tables generated by preprocessing scripts

## Rules

1. Raw files must remain immutable after download.
2. All transformations must happen through scripts in `src/`.
3. Processed datasets should be reproducible from raw inputs and configuration.
4. The analytical dataset should contain features needed for the F0 and F1 variants described in Assignment 3.

## Expected raw sources

- GP practice list sizes and demographics
- GP workforce / availability data
- hospital use / admissions outcome data
- optional regional or deprivation control variables
